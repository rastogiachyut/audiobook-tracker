📚 Audiobook Tracker

💾 Save to File

To save directly to data.json and source-data.js, you need the local server running.

1. Open a terminal and navigate to the project folder: cd ~/Projects/audiobook-tracker 2. Start the server: python3 server.py 3. Open in browser: http://localhost:8420
The server watches for POST /api/save and writes both files atomically.

💡 Tip: Once the server is running, the 💾 Save button in the header writes directly — no modal needed.
